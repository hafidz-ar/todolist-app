# Product: todolist-app

A simple client-side todo list web app. Users can add, complete, filter, and delete tasks through a responsive browser UI. No backend — all data persists via `localStorage`.

## Core Features
- Add tasks (text input, max 200 chars, trims whitespace)
- Toggle task completion (checkbox)
- Delete tasks
- Filter tasks: All / Active (incomplete) / Completed
- Task summary showing incomplete count
- Auto-save to `localStorage` on every state change

## Language
UI labels and documentation are written in **Bahasa Indonesia**.
